# 商用電商網站優化清單

> 日沐 SunBathe 電商平台 - 商業化優化評估報告

---

## 🔒 安全性 (最重要)

| 項目          | 狀態   | 說明                         |
| ------------- | ------ | ---------------------------- |
| HTTPS         | 需確認 | 生產環境必須使用 HTTPS       |
| XSS 防護      | ⚠️     | 確認所有 `v-html` 內容已消毒 |
| CSRF 防護     | 需確認 | API 是否有 CSRF token        |
| Rate Limiting | 需確認 | 防止暴力破解和濫用           |
| 敏感資料加密  | 需確認 | 信用卡等資料不可儲存在前端   |

---

## 📊 商業功能 (必備)

| 功能                   | 狀態    | 建議                     |
| ---------------------- | ------- | ------------------------ |
| Google Analytics / GA4 | ❌ 缺少 | 追蹤用戶行為、轉換率     |
| Facebook Pixel         | ❌ 缺少 | 廣告追蹤、再行銷         |
| 結帳轉換追蹤           | ❌ 缺少 | 購買完成事件追蹤         |
| 商品搜尋               | ⚠️ 基礎 | 需加強搜尋建議、全文搜索 |
| 商品篩選               | ⚠️      | 價格區間、分類、標籤篩選 |
| 收藏/願望清單          | ❌ 缺少 | 提升用戶黏著度           |
| 商品評價系統           | ❌ 缺少 | 增加購買信任度           |
| 優惠碼/折扣券          | ❌ 缺少 | 促銷必備功能             |
